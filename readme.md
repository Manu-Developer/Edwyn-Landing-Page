# Uso

Instalar dependencias (Tailwind)

```
npm install
```

Para compilar Tailwind

```
npm run build
```

Ejecute Tailwind CLI para compilar **input.css** durante el desarrollo. El archivo de salida es **css/main.css**

```
npm run watch
```
